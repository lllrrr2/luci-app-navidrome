## 简介

[中文文档](README.md) | [English](README_en.md)

在 OpenWrt 路由器上直接运行 Navidrome，而不是通过 Docker

## 功能

- [x] 运行 Navidrome
- [x] 二进制文件自动下载
- [x] 更多的设置项
- [x] 多语言支持
- [x] 日志显示
- [ ] 根据机型下载二进制文件

## 说明

- 当前为初始版本，可能会产生某些预期外的错误，谨慎使用
- 当前只支持 x86/64
- 如需要转码，请自行安装 ffmpeg（未测试）
